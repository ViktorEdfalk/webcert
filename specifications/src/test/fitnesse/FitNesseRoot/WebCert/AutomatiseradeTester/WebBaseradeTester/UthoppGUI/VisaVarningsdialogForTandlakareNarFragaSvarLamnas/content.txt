Testet kontroller att tandläkare som kommer in till Webcert via uthoppslänk får se en varningsdialog 
när vederbörande väljer att lämna sidan 'Frågor och svar' och gå till 'Ej signerade utkast' eller 'Sök/skriv intyg'

!3 Logga in

| skript | !-LoggaIn-! |
| logga in som | tore-tandlakare |

!3 Skapa ett nytt intygsutkast

| skript | !-SkrivIntyg-! |
| skapa nytt intygsutkast for patient | ${person1.personnummer} | av typ | FK7263 |
| $intygsid= | intygsid |


!3 Byt roll för att simulera att användare kommer via uthoppslänk

| skript | Anvandarroll |
| ändra till | ROLE_TANDLAKARE_UTHOPP |

!3 Visa utkastet via uthoppslänk

| skript | !-IntegrationViaUthoppslank-! |
| ga till intygsvy via uthopp med intygs id | $intygsid |
| säkerställ att | meddelande intyget inte skickat visas |

!3 Gå till Ej signerade utkast

| skript | !-IntegrationViaUthoppslank-! |
| ga till ej signerade utkast |
| säkerställ att | lamna fraga svar varning visas |
| avbryt lamna fraga och svar |

!3 Visa utkastet via uthoppslänk

| skript | !-IntegrationViaUthoppslank-! |
| ga till intygsvy via uthopp med intygs id | $intygsid |

!3 Gå till Sök/skriv intyg

| skript | !-IntegrationViaUthoppslank-! |
| ga till sok skriva intyg |
| säkerställ att | lamna fraga svar varning visas |
| avbryt lamna fraga och svar |

!3 Ta bort utkastet

| skript | !-TaBortUtkast-! |
| ta bort utkast | $intygsid |
