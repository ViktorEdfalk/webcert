Detta test ska visa på att man kan skicka ett signerat ts-intyg.

!3 Logga in

| skript | !-LoggaIn-! |
| logga in som | IFV1239877878-1049_IFV1239877878-1042 |

!3 Välj patient och visa intyget

| skript | !-SokSkrivIntyg-! |
| säkerställ att | sok skriv intyg sidan visas |
| valj patient | ${person1.personnummer} |
| säkerställ att | valj intygstyp sidan visas |
| visa intyg | intyg-fit-8 |
| säkerställ att | visa intyg sidan visas |

!3 Skicka intyget kontrollera att intyget skickats

| skript | !-SokSkrivIntyg-! |
| säkerställ att | meddelande for status intyg inte skickat visas | Transportstyrelsen |
| skicka det visade intyget av typ | ts-diabetes |
| säkerställ att | meddelande for status intyg skickat visas | Transportstyrelsen |


!3 Verififera att intyget finns i stubben

| ts intyg finns i stub |
| id | exists? |
| intyg-fit-8 | true |

| skript | !-IntygstjanstAsynk-! |
| säkerställ att | finns intyg | intyg-fit-8 | i intygstjansten skickad till | TS | med vantetid | 5000 |
