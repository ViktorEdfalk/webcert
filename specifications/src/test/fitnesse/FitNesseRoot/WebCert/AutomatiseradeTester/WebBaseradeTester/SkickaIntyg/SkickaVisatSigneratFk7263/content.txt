Detta test ska visa på att man kan skicka ett signerat fk-intyg.

!3 Logga in

| skript | !-LoggaIn-! |
| logga in som | IFV1239877878-1049_IFV1239877878-1042 |

!3 Välj patient och visa intyget

| skript | !-SokSkrivIntyg-! |
| säkerställ att | sok skriv intyg sidan visas |
| valj patient | ${person1.personnummer} |
| säkerställ att | valj intygstyp sidan visas |
| visa intyg | intyg-fit-6 |
| säkerställ att | visa intyg sidan visas |

!3 Skicka intyget och kontrollera att intyget skickats

| skript | !-SokSkrivIntyg-! |
| säkerställ att | meddelande intyg inte skickat visas |
| säkerställ att | meddelande for status intyg inte skickat visas | Försäkringskassans |
| skicka det visade intyget av typ | fk7263 |
| säkerställ att | meddelande for status intyg skickat visas | Försäkringskassans system |

!3 Kontrollera att Intygstjänsten (asynkront) tagit emot och skickat intyget (webcert-1314)

| skript | !-IntygstjanstAsynk-! |
| säkerställ att | finns intyg | intyg-fit-6 | i intygstjansten skickad till | FK | med vantetid | 5000 |
